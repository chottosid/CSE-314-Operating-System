# CSE-314 — Operating Systems Sessional

Repository of lab assignments for **CSE-314: Operating Systems (Sessional)**.
All tasks implemented and tested on **xv6**.

## Topics covered

* System calls
* Inter-process communication (IPC)
* Threading and synchronization
* Process scheduling and management
* Memory management experiments
* File system exercises
* Concurrency control (locks, semaphores, condition variables)
